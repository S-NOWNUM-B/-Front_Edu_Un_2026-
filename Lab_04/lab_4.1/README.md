# Lab 4.1: Controlled Forms with Real-Time Validation

## Обзор
Данная лабораторная работа сфокусирована на создании контролируемых форм (controlled forms) в React с использованием TypeScript. Реализована форма регистрации с валидацией полей в реальном времени и отображением ошибок.

## Цель работы
Научиться управлять состоянием форм через хуки React, реализовывать валидацию пользовательского ввода и обеспечивать мгновенную обратную связь для улучшения UX.

## Структура проекта в src:
*   `RegistrationForm.tsx` — Основной компонент формы регистрации. Содержит логику управления состоянием трех полей (Name, Email, Age), валидацию в реальном времени и обработку отправки формы.
*   `App.tsx` — Корневой компонент, который отображает форму регистрации с заголовком.

## Реализованный функционал
1.  **Controlled Inputs:** Каждое поле формы связано с собственным состоянием через `useState<string>`, что делает React единым источником истины (single source of truth).
2.  **Real-Time Validation:** Валидация срабатывает при каждом изменении поля (`onChange`), обеспечивая мгновенную обратную связь:
    *   **Name:** Минимум 2 символа
    *   **Email:** Проверка регулярным выражением на корректный формат email
    *   **Age:** Должно быть числом и не менее 18
3.  **Error State Management:** Отдельное состояние `errors` хранит сообщения об ошибках для каждого поля. Ошибки отображаются красным текстом под соответствующим полем.
4.  **Submit Handler:** При отправке формы (`onSubmit`) выполняется финальная валидация. Если все поля корректны, отображается сообщение "Registration successful!" зеленым цветом.
5.  **TypeScript Integration:** Использование интерфейсов (`ErrorState`) для строгой типизации состояния ошибок.

## Методология
Подход основан на концепции контролируемых компонентов, где React полностью управляет состоянием формы. Это обеспечивает предсказуемость поведения, упрощает валидацию и позволяет легко добавлять дополнительную логику (например, отправку на сервер).
