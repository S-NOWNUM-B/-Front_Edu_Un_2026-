# Lab 11.1: Основы React Native и стилизация компонентов

## Обзор
Данная лабораторная работа посвящена изучению базовых принципов создания мобильных приложений с использованием React Native и Expo. Основная цель — научиться создавать композицию UI-компонентов и применять к ним стили через `StyleSheet`.

## Цель работы
Освоить создание базовых компонентов (`View`, `Text`, `Image` и др.), работу с Flexbox в React Native, а также вынос логики и интерфейса в переиспользуемые блоки.

## Структура проекта в src/components:
Согласно бизнес-логике, интерфейс поделен на компоненты:
*   `ProfileCard.tsx` — карточка профиля пользователя с изображением и базовой информацией.
*   `ContactSection.tsx` — секция с контактными данными и блоками информации.
*   `App.tsx` — корневой файл для отрисовки интерфейса.

## Реализованные задачи
1.  **Базовая верстка (Layout):** Реализована семантическая компоновка элементов интерфейса через `View`.
2.  **Стилизация (Styling):** Успешно применен механизм `StyleSheet.create` для описания свойств (отступы, цвета, шрифты, flex-позиционирование).
3.  **Изоляция компонентов:** Код разбит на отдельные модули для переиспользования и удобства поддержки мобильного приложения.

## Инструкция по запуску
1. Установите зависимости: `npm install`
2. Запустите сервер разработки Expo: `npx expo start`
3. Отсканируйте QR-код в приложении **Expo Go** на вашем смартфоне или запустите в эмуляторе (Android/iOS).
