# Lab 11.2: Навигация в React Native (React Navigation)

## Обзор
В данной лабораторной работе фокус смещается на управление переходами между различными экранами в приложении. Основная цель — настроить стек-навигацию (Stack Navigation) и научиться передавать данные от одного экрана к другому.

## Цель работы
Освоить популярную библиотеку `@react-navigation/native` и `@react-navigation/native-stack` для создания многостраничных мобильных приложений на базе Expo.

## Структура проекта в src:
Логика разбита по слоям в директориях экранов и данных:
*   `App.tsx` — инициализация `NavigationContainer` и `Stack.Navigator`, объявление маршрутов приложения (`ProductList` и `ProductDetail`).
*   `src/screens/ProductListScreen.tsx` — главный экран (каталог) со списком продуктов. Использует `FlatList` или `map` для отрисовки.
*   `src/screens/ProductDetailScreen.tsx` — экран с детальным описанием конкретного продукта.
*   `src/data/products.ts` — статические (моковые) данные для отображения списка.

## Реализованные задачи
1.  **Внедрение Stack-навигатора:** Настроены переходы между каталогом `ProductList` и экраном детализации `ProductDetail` с использованием компонента навигации.
2.  **Передача параметров:** Выполнено прокидывание объекта `product` в `route.params` при нажатии на элемент каталога, чтобы целевой экран корректно отрисовывал информацию.
3.  **Оформление Header:** Заданы стили для заголовка навигации (`headerStyle`, `headerTintColor`) и реализована автоматическая подстановка имени товара в title на детальном экране.

## Инструкция по запуску
1. Установите зависимости: `npm install`
2. Запустите сервер разработки Expo: `npx expo start -c`
3. Отсканируйте QR-код в свежей версии **Expo Go** на вашем смартфоне. Убедитесь, что SDK в приложении совпадает с версией проекта.
