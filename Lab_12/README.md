# Lab 12: Адаптивный интерфейс в React Native

## Обзор
Данная лабораторная работа посвящена построению адаптивного мобильного интерфейса в React Native/Expo с использованием переиспользуемых UI-компонентов и навигации между экранами.

## Цель работы
Освоить практику создания адаптивных компонентов интерфейса, их интеграции в единый экран и подключения в стек-навигацию приложения.

## Структура проекта в src:
Логика приложения разделена на экраны, компоненты и данные:
*   `App.tsx` — корневой файл приложения, настройка `NavigationContainer` и `Stack.Navigator`.
*   `src/screens/ProductListScreen.tsx` — экран каталога товаров с переходом к деталям и демо-компонентам.
*   `src/screens/ProductDetailScreen.tsx` — экран детального просмотра товара.
*   `src/screens/ComponentsShowcaseScreen.tsx` — демонстрационный экран, где подключены все кастомные компоненты.
*   `src/components/ProfileCard.tsx` — карточка профиля.
*   `src/components/ContactSection.tsx` — контактная секция с формой.
*   `src/components/ResponsiveHeader.tsx` — адаптивный заголовок и контейнер.
*   `src/components/GridLayout.tsx` — сеточный layout и карточки.
*   `src/components/AdaptiveLayout.tsx` — адаптивная компоновка контента, `FeatureCard`, `StatsRow`.
*   `src/components/ProductItem.tsx` — карточка товара в списке.
*   `src/data/products.ts` — моковые данные каталога.

## Реализованные задачи
1.  **Адаптивная композиция UI:** Подготовлены и подключены компоненты для мобильного и tablet-сценария отображения.
2.  **Переиспользуемые блоки интерфейса:** Реализованы отдельные компоненты (`ProfileCard`, `ContactSection`, `GridLayout`, `ResponsiveHeader`, `AdaptiveLayout`) и объединены на демо-экране.
3.  **Навигация между экранами:** Настроены переходы между каталогом, деталями товара и экраном демонстрации компонентов.
4.  **Совместимость с web в Expo:** Добавлены необходимые зависимости для web-платформы и проверена успешная сборка.

## Инструкция по запуску
1. Установите зависимости: `pnpm install`
2. Запустите проект: `npx expo start -c`
3. Для web-проверки выполните: `pnpm exec expo export --platform web`
4. Откройте экран `Components Demo` с главного экрана, чтобы увидеть подключение всех компонентов.
