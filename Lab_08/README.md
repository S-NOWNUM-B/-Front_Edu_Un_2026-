# Lab 08

В этой лабораторной используются три подхода к рендерингу в Next.js:

- **SSR (Server-Side Rendering):** данные всегда свежие, потому что страница генерируется на сервере при каждом запросе. Подходит для личных кабинетов, дашбордов и другого персонализированного контента.
- **SSG (Static Site Generation):** самая быстрая загрузка, потому что страница генерируется один раз во время сборки проекта. Подходит для блогов, лендингов и статических страниц.
- **ISR (Incremental Static Regeneration):** компромиссный подход - страница остается статической, но обновляется в фоне через заданный интервал `revalidate`.
